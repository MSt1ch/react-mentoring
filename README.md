# react-mentoring

## Install and run

In the root directory type you need write commands:

```
yarn
yarn start
```

and open [http://localhost:3000].

---
To clean before production:
```
yarn clean
```
To start test:
```
yarn test 
```
To start cypress:
```
yarn cypress 
```
To see coverage:
```
yarn coverage 
```
---
To deploy production:
```
yarn build-prod
```
